# RFQ Auction System

**Overview**
A real-time RFQ auction system where suppliers continuously underbid each other with dynamic auction extension logic.

**Tech Stack**
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL (Prisma)
- Realtime: Socket.io

**Features**
- RFQ creation with scheduling
- Real-time bidding
- Dynamic auction extension
- Supplier ranking (L1, L2, L3)
- Activity logs with extension reasons

**How to Run**
Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm start
